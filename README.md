AI Resume Analyzer

Overview

AI Resume Analyzer is a Python and Streamlit-based web application that analyzes resumes and compares them with job descriptions.

Features

- Upload Resume PDF
- Extract Resume Text
- Detect Technical Skills
- Job Description Matching
- Match Score Calculation
- Missing Skills Identification

Technologies Used

- Python
- Streamlit
- PyPDF2
- Pandas

Project Structure

- app.py
- skills.py
- requirements.txt
- screenshots/

How to Run

pip install -r requirements.txt
python3 -m streamlit run app.py
